10 /* This header file contains classes implementing RenderWare Binary Stream File nodes.
11 * The following inheritance structure is used:
12 * RWChunk ~~~~~~~~~~~~~~~~~~~ Implements the section ID and the library ID.
13 * | ~ Can be queried for both and implements a helper function
14 * | ~ to identify if it's a container or not.
15 * |-- RWContainer ~~~~~~~~~~~ Receives a char pointer to a data array that can later be used to "unfold"
16 * | | ~ the tree, i.e. create children from the data.
17 * | | ~ Those children can then be accessed directly through the
18 * | | ~ array accessor operator.
19 * | |--> Implementations
20 * |
21 * |-- RWData ~~~~~~~~~~~~~~~~ Also receives a char pointer to a data array that is, however, meant to
22 * | | ~ be used in conjunction with data conversion functions
23 * | | ~ implemented statically in the RWChunk namespace.
24 * | |--> Implementations
25 * |
26 * |-- RWInvalid ~~~~~~~~~~~~~ Meant to be used if an invalid chunk structure is detected.
27 * | ~ Similar to RWData, but the data array shall contain the
28 * | ~ supposed "header" data.
29 * |--> Implementations
30 */
